2､什麼是軟件工程師呢？
	是一種職位的名稱
	通常是通過計算機的某種編程語言完成軟件的開發
3､什麼是軟件呢？

	大家眼前的筆記本電腦就是一台計算機

	計算機包括兩部分：

		* 硬件：鼠標鍵盤､顯示器､主機箱內部的CPU､內存條､硬盤等
		注意､計算機只有硬件是無法工作的，需要軟件驅動硬件才能工作。

		*軟件：
			軟件包括：系統軟件和應用軟件
			系統軟件：
				直接和硬件交互的軟件，例如： window7, winxp, win8, win10, Linux 操作系統
			應用軟件：
				應用軟件通常運行在系統軟件中，例如：QQ運行在windows操作系統上。
				QQ就是應用軟件。
				win7就操作系統

我們通常所說的軟件開發一般都是指使用計算機編程語言完成"應用軟件'"的開發

7､作為程序員要求掌握最基本的windows相關Dos命令：
	* DOS 命令在哪裡編寫？
		在DOS命令窗口當中編寫DOS命令
	* DOS命令窗口怎麼打開呢？
		開始菜單-->運行-->cmd-->回車
	* 常見的DOS命令？
	- exit 退出當前DOS命令窗口
	- cls 清屏
	- DOS 窗口的任意位置，點擊鼠標右鍵，然後選擇標記，
	  接下來選中要複製的內容，在DOS窗口的任意位置點擊鼠標右鍵，
	  此時被複製的內容己經到剪貼板了。
	  在需要黏貼的位置黏貼即可。
	- dir 列出當前目錄下所有的子文件/子目錄
	- cd 命令
		* cd命令表示： change directory 改變目錄
		* cd命令怎麼用呢？
			cd 目錄的路徑
		* 但是路徑包括絕對路徑和相對路徑
		* 絕對路徑：表示該路徑從某個磁盤的盤符下作為出發點的路徑
		* 相對路徑：表示該路徑從當前所在的路徑下作為出發點的路徑
	- cd .. 回到上級目錄。
	- cd \ 直接回到根目錄
	-怎麼切換盤符：
	  c:
	  d:
	  e:
	  f:

8､關於windows操作系統的文件擴展名的設置：
	作為java程序員，以後要新建.java結尾的文件，所以必須保證文件的擴展名是.java
	作麼做呢？
		打開計算機-->組織-->文件夾和搜索選項-->隱藏己知文件類型的擴展名（前面對勾去掉）
